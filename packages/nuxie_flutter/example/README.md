# nuxie_flutter_example

Reference app for the Nuxie Flutter wrapper.

## What It Covers

- `Nuxie.initialize(...)`
- `identify(...)`
- `trigger(...)` and `triggerOnce(...)`
- `showFlow(...)`
- `hasFeature(...)`
- Dart-side purchase and restore controller callbacks

## Run

```bash
flutter pub get
flutter run
```
